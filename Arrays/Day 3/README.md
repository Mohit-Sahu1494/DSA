## 💡 Key Learning

- Sliding Window is used to optimize subarray and substring problems.
- Instead of recalculating every window, update the current window by adding the incoming element and removing the outgoing element.
- Fixed Window is used when the window size is constant.
- Variable Window is used when the window size depends on a condition.
- HashSet is useful for unique element problems.
- HashMap is useful when frequency counting is required.

## ⚡ Time Complexity
